# Распоред часова

Софтвер за аутоматску израду распореда часова за основне и средње школе.

## Функционалности

### 📋 Унос података
- **Подаци о школи**: Име школе, тип (основна/средња), број смене, врста часова
- **Одељења**: Унос броја одељења по разредима
- **Предмети**: Додавање предмета са бројем часова недељно
- **Наставници**: Додела наставника предметима
- **Кabinети**: Опционо додељивање кабинета предметима

### 🎯 Оптимизација распореда
- **Балансирање**: Разврставање предмета по категоријама (природни, друштвени, језици, уметност, спорт)
- **Минимизација оптерећења**: Ученици имају оптималан број часова дневно
- **Избегавање груписања**: Спречава груписање истих типова предмета у једном дану
- **Ефикасност**: Алгоритам оптимизује распоред за све учеснике

### 📊 Прикази распореда
- **По одељењима**: Преглед распореда за свако одељење
- **По наставницима**: Распоред часова за сваког наставника
- **По кабинетима**: Заузеће кабинета по часовима

### 🎨 Дизајн
- **Модерни UI**: Професионалан изглед прилагођен школству
- **Responsivan**: Ради на свим уређајима
- **Интуитиван**: Једноставно коришћење за све кориснике

## Коришћење

1. **Покретање апликације**
   ```bash
   npm install
   npm start
   ```

2. **Креирање распореда**
   - Унесите основне податке о школи
   - Додајте одељења и предмете
   - Доделите наставнике и кабинете
   - Кликните на "Генериши распоред"

3. **Преглед распореда**
   - Изаберите жељени приказ (одељења/наставници/кabinети)
   - Селектујте конкретно одељење/наставника/кabinet
   - Прегледајте генерисани распоред

## Технологије

- **React 18** - Frontend framework
- **TypeScript** - Тип сигуран JavaScript
- **Tailwind CSS** - Модерни CSS framework
- **Lucide React** - Иконе

## Структура пројекта

```
src/
├── components/          # React компоненте
│   ├── SchoolForm.tsx   # Форма за унос података о школи
│   ├── ClassesForm.tsx  # Форма за одељења и предмете
│   └── ScheduleView.tsx # Приказ распореда
├── types/              # TypeScript типови
│   └── index.ts        # Дефиниције типова
├── utils/              # Помоћне функције
│   └── scheduleGenerator.ts # Алгоритам за генерисање распореда
├── App.tsx             # Главна апликација
└── index.tsx           # Entry point
```

## Алгоритам оптимизације

Софтвер користи напредан алгоритам који:
- Разврстава предмете по категоријама
- Балансира дневно оптерећење
- Избегава сукобе наставника и кабинета
- Минимизује празне сате у распореду
- Оптимизује распоред за све учеснике
